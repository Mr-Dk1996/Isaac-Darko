Isaac Darko Multi-Page Portfolio Update

Files included:
- index.html       Homepage with hero, snapshot, and featured project
- about.html       About page
- experience.html  Combined Experience and Skills page
- contact.html     Contact page with Netlify-compatible form
- styles.css       Main styling
- mediaqueries.css Mobile responsiveness
- script.js        Mobile menu and footer year

How to update in VS Code:
1. Open your existing portfolio folder in VS Code.
2. Make a backup of your current files first.
3. Copy these new files into the same project folder where your current index.html is.
4. Allow VS Code/Windows to replace index.html, styles.css, mediaqueries.css, and script.js.
5. Add about.html, experience.html, and contact.html to the same folder.
6. Keep your existing assets folder because the pages still use assets/profile-pic.jpg, assets/about-pic.jpg, assets/project-1.png, and assets/resume.pdf.
7. Delete or ignore the old form.html and gallery.html if you no longer want them in the website navigation.
8. Test locally by opening index.html in the browser or using Live Server.
9. Push/deploy to Netlify.

Important:
Netlify does not run submit_form.php on a normal static site. The new contact.html uses a Netlify form instead.
